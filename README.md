# AquaNet Generation Scripts

*Please* read the comments in the scripts before you use them.<br>
Anyone is welcome to add to this list.

all-music.json should be placed as `(data base)/(game)/00/all-music.json`<br>
all-items.json should be placed as `(data base)/(game)/00/all-items.json`<br>
Jackets should be placed as `(data base)/(game)/music/(6-digit id).png`

Game identifiers:
 - `mai2` => Maimai (SDEZ/SDGA)
 - `chu3` => Chusan (SDHD)
 - `wacca` => Wacca
 - `ongeki` => Ongeki
